# House-Price-Prediction
Simple house price prediction using python
This project predicts house prices using a simple machine learning model.

## Files in this Project
- house_price.py – Python backend and ML model
- house.csv – Dataset
- index.html – Frontend (HTML)
- style.css – Styling (CSS)

## Technologies Used
- Python
- Pandas
- NumPy
- scikit-learn
- HTML
- CSS

## How to Run
1. Install required libraries:
   pip install pandas numpy scikit-learn
2. Run Python file:
   python app.py
3. Open index.html in browser for UI demo

## Description
This is a beginner-friendly ML project suitable for college presentations.
